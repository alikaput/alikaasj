Pengertian Hosting

Hosting adalah sebuah layanan online yang digunakan untuk mengelola data situs atau aplikasi web oleh pengguna dan ditampilkan melalui via internet.



Fungsi Hosting

1. Membuat Halaman Website
2. Mengembangkan Situs Jual Beli Online
3. Mampu Melindungi Website dengan SSL



Jenis-Jenis Hosting

1. Cloud Based Hosting
2. Dedicated Hosting
3. Shared Hosting
